## Thème

Ce projet une conecption et une réalisation d'un site web de réservation de voyage

## Technoogies
- Frontend : HTML, CSS, JS et Bootstrap
- Backend : Node.js, express
- Base de données : MariaDB (MySQL compatible)

## Fonctionnalités
- Partie client :
	- Accueil
	- Réservation
	- Sélection de siège
	- Confiramation
	- Historique

- Partie administration :
	- Dashboard admin
	- Gestion
